# 模板与 AI 辅助

## 内置模板

演讲宝提供了多套精美的 PPTE 幻灯片模板，让你无需从零开始设计。

### 使用模板创建幻灯片

1. 在**课程管理**界面中，找到你想添加幻灯片的章节
2. 点击 **创建幻灯片** 按钮
3. 输入幻灯片组的名称
4. 系统会使用内置模板自动生成一组基础幻灯片
5. 你可以在此基础上进行编辑和定制

### 模板包含的页面类型

每套模板通常包含以下页面类型：

| 类型 | 说明 | 用途 |
|------|------|------|
| **封面** (Cover) | 标题页，展示主题和演讲者信息 | 演示的第一页 |
| **目录** (Catalog) | 大纲页，列出各章节标题 | 帮助听众了解结构 |
| **章节** (Chapter) | 章节分隔页，标记新章节开始 | 章节间过渡 |
| **内容** (Content) | 正文页，承载具体内容 | 核心内容展示 |
| **结尾** (Finish) | 致谢页，总结和联系方式 | 演示的最后一页 |

## 编辑 PPTE 幻灯片

### 进入编辑模式

1. 在课程管理界面中，找到已有的 PPTE 资源
2. 点击 **编辑** 按钮
3. 侧边栏会自动隐藏，提供更大的编辑空间
4. 编辑完成后关闭编辑器，侧边栏自动恢复

### 编辑操作

在 PPTE 编辑器中，你可以：

- **添加新页面** — 从模板中选择页面类型添加
- **删除页面** — 移除不需要的页面
- **调整顺序** — 拖拽页面缩略图改变顺序
- **编辑内容** — 直接修改 HTML 代码
- **实时预览** — 边编辑边查看效果

## AI 辅助编辑

### AI 幻灯片助手

演讲宝提供 AI 辅助入口。公开版需要你在设置中配置自己的 DeepSeek 或 MiniMax API Key，才能使用 AI 创建和编辑 PPTE 幻灯片。

### 使用方法

1. 在 PPTE 编辑模式中，打开右侧的 **AI 对话面板**
2. 用自然语言描述你想要的修改
3. AI 会生成对应的 HTML/CSS 代码
4. 确认后自动应用到当前页面

### 有效的提示示例

以下是一些能产生良好结果的 AI 提示示例：

#### 修改内容

```
请将标题改为"深度学习基础"，副标题改为"从零开始的神经网络之旅"
```

```
在内容区域添加三个要点：
1. 数据预处理的重要性
2. 模型选择与超参数调优
3. 评估指标与验证方法
```

#### 修改样式

```
将背景色改为深蓝色渐变，文字改为白色
```

```
给标题添加一个从左到右的渐变色效果，从蓝色到紫色
```

#### 生成新页面

```
创建一个内容页，主题是"机器学习工作流程"，包含流程图：
数据收集 → 数据清洗 → 特征工程 → 模型训练 → 评估 → 部署
```

### AI 使用提示

- **具体描述**：越具体的描述越容易得到满意的结果
- **分步修改**：复杂的修改建议分多次进行
- **保留预览**：修改后先预览效果，再确认应用
- **迭代优化**：对 AI 的结果不满意时，可以继续提出修改要求

## 手动编辑

如果你熟悉 HTML 和 CSS，也可以直接编辑 PPTE 文件：

1. 找到课程目录下的 PPTE 文件夹
2. 使用任何文本编辑器打开 HTML 文件
3. 修改内容和样式
4. 保存后在演讲宝中刷新查看效果

### 常用 CSS 技巧

```css
/* 渐变背景 */
background: linear-gradient(135deg, #1a1a2e 0%, #16213e 50%, #0f3460 100%);

/* 文字阴影 */
text-shadow: 0 2px 4px rgba(0, 0, 0, 0.3);

/* 卡片效果 */
border-radius: 12px;
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.2);

/* 渐变文字 */
background: linear-gradient(90deg, #00d2ff, #7b2ff7);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

## 最佳实践

1. **一页一主题** — 每页幻灯片只讲一个核心要点
2. **少文字多图示** — 幻灯片是视觉辅助工具，不是文档
3. **统一风格** — 同一组幻灯片保持一致的配色和字体
4. **善用 AI** — 让 AI 处理样式和排版，你专注于内容
5. **反复预览** — 每次修改后都预览效果，确保显示正常
