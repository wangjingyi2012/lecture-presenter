# 支持的资源类型

## 概述

演讲宝内置了多种专业查看器，让你无需切换应用即可浏览几乎所有常见的教学资源格式。

---

## 1. PDF 查看器

**适用格式**：`.pdf` 文件

演讲宝使用 PDF.js 引擎提供高质量的 PDF 渲染体验。

### 功能特性

- 清晰的文本和矢量图形渲染
- 支持缩放（`Cmd+=` 放大，`Cmd+-` 缩小，`Cmd+0` 重置）
- 页码导航
- 滚动浏览

### 使用提示

- PDF 文件推荐分辨率为 150 DPI 以上，以获得最佳显示效果
- 大型 PDF 文件（超过 50MB）加载可能需要几秒钟

---

## 2. 视频播放器

**适用格式**：`.mp4` 文件

内置视频播放器支持标准的 HTML5 视频控制。

### 功能特性

- 播放 / 暂停控制
- 进度条拖拽
- 音量调节
- 全屏播放

### 使用提示

- 推荐使用 MP4 (H.264) 格式，兼容性最佳
- 其他视频格式（如 WebM、MOV）可能无法正常播放
- 视频不会自动播放，需手动点击播放按钮

---

## 3. Markdown 渲染器

**适用格式**：`.md` 文件

Markdown 文件会被自动渲染为美观的富文本页面。

### 支持的语法

- **标题**：`# 一级标题` 到 `###### 六级标题`
- **文本样式**：`**粗体**`、`*斜体*`、`~~删除线~~`
- **列表**：有序列表和无序列表
- **链接和图片**：`[文字](URL)` 和 `![描述](图片路径)`
- **代码块**：支持语法高亮的代码块
- **表格**：标准 Markdown 表格语法
- **引用**：`> 引用文字`
- **分隔线**：`---`

### 使用提示

- 你现在正在阅读的就是一个 Markdown 文件
- Markdown 文件非常适合用作教学文档、笔记和说明

---

## 4. 代码查看器

**适用格式**：`.py`、`.js`、`.ts`、`.java`、`.cpp`、`.go`、`.rs`、`.html`、`.css`、`.json`、`.yaml`、`.toml`、`.sh`、`.sql` 等

代码文件使用 Highlight.js 引擎进行语法高亮显示。

### 功能特性

- 自动语言检测
- 语法高亮（支持数十种编程语言）
- 行号显示
- 适配浅色/深色主题

### 使用提示

- 代码文件以只读模式打开
- 支持的语言类型非常广泛，几乎涵盖所有主流编程语言

---

## 5. HTML 查看器

**适用格式**：`.html`、`.htm` 文件

HTML 文件使用内置的 WebView 进行渲染，可以展示完整的网页内容。

### 功能特性

- 完整的 HTML/CSS/JS 渲染
- 支持内联样式和外部样式表
- 支持 JavaScript 执行

### 使用提示

- HTML 文件中的外部资源（图片、样式等）使用相对路径引用
- 出于安全考虑，某些功能可能受到限制

---

## 6. PPT-EXTRA 幻灯片

**适用格式**：包含 `manifest.json` 的 PPTE 目录

PPT-EXTRA 是演讲宝独创的幻灯片格式，使用 HTML 文件模拟 PPT 翻页效果。

### 功能特性

- 16:9 标准比例的幻灯片显示
- 左右翻页导航
- 页码显示
- 支持全屏演示
- 丰富的视觉效果（渐变、动画、自定义排版）

### 使用提示

- 点击页面左侧/右侧切换上一页/下一页
- 也可使用左右方向键翻页
- PPTE 幻灯片可以在课程创建器中创建和编辑

> 关于 PPTE 格式的详细说明，请参阅 **第四章：PPTE 幻灯片**。

---

## 7. 外部链接（URL）

**行为**：在系统默认浏览器中打开

外部链接类型的资源会在你的默认浏览器中打开，适用于在线文档、网站、视频平台等。

### 使用提示

- 确保有网络连接才能访问在线资源
- 外部链接的图标会有特殊标记，与本地文件区分

---

## 资源类型总结

| 类型 | 文件格式 | 查看方式 |
|------|----------|----------|
| PDF | `.pdf` | 内置 PDF.js 查看器 |
| 视频 | `.mp4` | 内置 HTML5 播放器 |
| Markdown | `.md` | 内置 Markdown 渲染器 |
| 代码 | `.py`、`.js` 等 | 语法高亮查看器 |
| HTML | `.html` | 内置 WebView |
| PPTE | `manifest.json` 目录 | PPTE 幻灯片查看器 |
| URL | 网址 | 系统默认浏览器 |
| 目录 | 文件夹 | Finder |
