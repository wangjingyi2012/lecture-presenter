# AI 助手配置

## 简介

演讲宝公开版支持使用你自己的 AI API Key 来辅助编辑 PPTE 幻灯片。当前保留 DeepSeek 和 MiniMax 两种提供商入口。

## 如何配置

1. 打开右上角设置。
2. 找到 **AI 配置**。
3. 选择 DeepSeek 或 MiniMax。
4. 输入你自己的 API Key。
5. 回到 PPTE 编辑器，打开 AI 对话面板。

API Key 会保存在本机应用配置中，请不要提交到 Git 仓库。

## 可做什么

- 根据自然语言修改当前 HTML 幻灯片。
- 调整标题、配色、排版和内容结构。
- 生成新的页面草稿。
- 帮助润色演示文案。

## 使用建议

- 描述具体目标，例如“把标题改成深度学习入门，并使用蓝紫色渐变背景”。
- 一次只改一类问题，复杂页面分多轮调整。
- 应用 AI 修改前先预览，必要时手动微调 HTML/CSS。

## 常见问题

### Q: 公开版自带托管 AI 服务吗？

不自带。公开仓库只发布客户端代码，托管 AI、账号、会员等服务端能力不包含在本仓库中。

### Q: API Key 会上传到哪里？

不会主动上传到项目仓库。它会存储在本机应用配置里，并在调用你选择的 AI 提供商时使用。

### Q: 可以不用 AI 吗？

可以。PPTE 幻灯片本质上是 HTML/CSS 文件，你可以完全手动编辑。
