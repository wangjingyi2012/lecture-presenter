# LectureAI 与会员

## LectureAI 简介

**LectureAI** 是演讲宝内置的 AI 智能助手，它可以帮助你：

- **编辑 PPTE 幻灯片** — 用自然语言描述修改需求，AI 自动生成代码
- **优化内容** — 润色文案、调整排版、改进视觉效果
- **回答问题** — 解答关于演讲宝使用的各种问题

---

## 如何使用

### 打开 AI 对话窗口

1. 点击标题栏右侧的 **AI 助手按钮**
2. 在弹出的对话窗口中输入你的问题或请求
3. AI 会根据当前上下文给出回复

### 在 PPTE 编辑中使用

1. 进入 PPTE 幻灯片编辑模式
2. 打开 AI 对话面板
3. 描述你想要的修改（如"把背景改成蓝色渐变"）
4. AI 生成修改方案后，确认应用

## 会员等级与权益

演讲宝提供多个会员等级，满足不同用户的需求。

### 等级对比

| 功能 | 基础版 (Basic) | 专业版 (Pro) | 旗舰版 (Ultra) |
|------|---------------|-------------|----------------|
| 课程导入与浏览 | 不限 | 不限 | 不限 |
| PPTE 幻灯片查看 | 不限 | 不限 | 不限 |
| PPTE 幻灯片编辑 | 不限 | 不限 | 不限 |
| **LectureAI 每日额度** | **10 次** | **50 次** | **100 次** |
| 模板数量 | 基础模板 | 全部模板 | 全部模板 |
| 优先支持 | - | 邮件支持 | 专属支持 |

### 每日额度说明

- LectureAI 的使用次数每天重置（北京时间 00:00）
- 每次对话交互（发送一条消息并收到回复）计为一次使用
- 额度用完后，当天无法继续使用 AI 功能，但其他功能不受影响

---

## 自配置 AI

如果你拥有自己的 AI 服务 API Key，也可以在设置中选择 DeepSeek 或 MiniMax 并填写 API Key。

> **提示**：自配置 AI 的费用由你的 API 服务商直接计费。
