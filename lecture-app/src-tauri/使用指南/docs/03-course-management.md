# 导入与管理课程

## 什么是"课程"？

在演讲宝中，一个**课程**是一个包含 `course.json` 配置文件的文件夹。该文件夹可以放在磁盘的任意位置，应用通过绝对路径访问其中的资源。

课程文件夹的推荐结构如下：

```
我的课程/
├── course.json          # 必须 — 课程元数据与内容描述
├── slides/              # 课件（PDF 文件）
├── videos/              # 视频（MP4 文件）
├── pdfs/                # 阅读材料（PDF 文件）
├── assignments/         # 作业目录
├── source_code/         # 示例代码
├── docs/                # Markdown 文档
└── ppt-extra/           # PPTE 幻灯片
    └── chapter01/
        ├── manifest.json
        ├── slide01.html
        └── slide02.html
```

> **提示**：子目录名称可以自由命名，`course.json` 中的所有路径都是相对于课程根目录的。

## 导入课程

### 步骤

1. 点击标题栏中的 **`+`** 按钮
2. 在弹出的文件夹选择对话框中，选择包含 `course.json` 的课程根目录
3. 应用会自动读取并验证 `course.json`
4. 导入成功后，课程将出现在课程切换下拉框中

### 导入要求

- 所选目录下**必须存在** `course.json` 文件
- `course.json` 必须是合法的 JSON 格式
- `course.json` 中必须包含 `id` 字段
- `id` 字段不得与已导入的课程重复

### 导入失败排查

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 找不到课程配置 | 目录中没有 `course.json` | 确认选择了正确的目录 |
| JSON 格式错误 | `course.json` 语法有误 | 检查 JSON 格式，注意逗号和引号 |
| 课程 ID 重复 | 已存在相同 ID 的课程 | 修改 `course.json` 中的 `id` |

## 课程管理

点击标题栏右侧的 **设置按钮** → **管理课程**，进入课程管理界面。

### 可用操作

| 操作 | 说明 |
|------|------|
| **导入课程** | 添加新的课程文件夹 |
| **删除课程** | 从列表中移除课程（不会删除磁盘上的文件） |
| **编辑课程** | 修改课程标题和其他信息 |
| **调整顺序** | 拖拽课程卡片改变显示顺序 |

> **注意**：删除课程只是从应用的列表中移除引用，磁盘上的课程文件夹不会受到任何影响。你可以随时重新导入。

## 创建新课程

### 使用内置课程创建器

演讲宝提供了可视化的课程创建工具：

1. 在课程管理界面中，点击 **创建课程** 按钮
2. 填写课程基本信息（名称、副标题、讲师）
3. 添加章节，并为每个章节配置资源
4. 保存后，应用自动生成 `course.json` 和目录结构

### 手动创建

你也可以手动创建课程：

1. 创建一个新的文件夹
2. 在文件夹中创建 `course.json` 文件
3. 按照格式规范编写课程配置（详见下一节）
4. 将资源文件（PDF、视频等）放入对应的子目录
5. 通过 `+` 按钮导入课程

## 切换课程

在标题栏的课程切换下拉框中选择你想浏览的课程即可完成切换。当前选中的课程名称会显示在下拉框中。

## 最佳实践

1. **统一命名规范** — 使用有意义的文件名，如 `week01-introduction.pdf` 而非 `file1.pdf`
2. **合理组织目录** — 将不同类型的资源放在对应的子目录中
3. **保持路径一致** — `course.json` 中的路径要与实际文件位置一致
4. **定期备份** — 重要的课程资料建议定期备份
5. **避免特殊字符** — 文件和目录名中避免使用特殊字符（空格除外）
