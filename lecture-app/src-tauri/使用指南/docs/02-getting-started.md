# 安装与首次启动

## 系统要求

- **操作系统**：macOS 10.15 (Catalina) 及以上版本
- **处理器**：Intel 或 Apple Silicon (M1/M2/M3/M4)
- **磁盘空间**：至少 200MB 可用空间

---

## 安装步骤

### 方式一：拖拽安装（推荐）

1. 双击下载的 `.dmg` 安装文件
2. 在弹出的窗口中，将 **Lecture Presenter** 图标拖拽到 **Applications** 文件夹
3. 等待复制完成，关闭安装窗口
4. 从"启动台"或"应用程序"文件夹打开 Lecture Presenter

### 首次打开提示

如果 macOS 提示"无法验证开发者"：

1. 打开 **系统偏好设置** → **安全性与隐私**
2. 点击"仍要打开"按钮
3. 在确认对话框中点击"打开"

> **提示**：这是 macOS 对非 App Store 应用的常规保护机制，演讲宝是安全的。

---

## 界面概览

首次启动后，你将看到演讲宝的主界面，由以下几个区域组成：

### 1. 标题栏（顶部）

标题栏包含以下功能区域：

| 区域 | 功能 |
|------|------|
| 课程切换下拉框 | 选择要浏览的课程 |
| 导入课程按钮 `+` | 导入新的课程文件夹 |
| 资源类型标签 | 课件、视频、阅读、作业、代码 等分类 |
| AI 助手按钮 | 打开 LectureAI 对话窗口 |
| 设置按钮 | 主题、字号等设置 |

### 2. 侧边栏（左侧）

- 显示当前课程的所有章节
- 点击章节名称即可跳转
- 使用 `Cmd+数字` 快捷键快速跳转（如 `Cmd+1` 跳转到第一章）
- 显示课程名称和讲师信息

### 3. 内容区域（中央）

- 主要的内容展示区域
- 根据资源类型自动选择合适的查看器
- PDF 文件使用内置 PDF 查看器
- 视频文件使用内置播放器
- Markdown 文件自动渲染为富文本
- 代码文件使用语法高亮查看器

---

## 基本导航

### 切换章节

- **方式一**：点击左侧边栏中的章节名称
- **方式二**：使用快捷键 `Cmd+1` 到 `Cmd+9`

### 切换资源类型

点击标题栏中的资源类型标签：

- **课件** — 幻灯片和演示文稿
- **视频** — 教学视频
- **阅读** — 阅读材料和参考文档
- **作业** — 课后练习和项目
- **代码** — 示例源代码

### 查看资源

- 点击资源列表中的项目名称即可打开
- 文件类型资源（PDF、视频、代码等）在内容区域中展示
- URL 类型资源在系统默认浏览器中打开
- 文件夹类型资源在 Finder 中打开
