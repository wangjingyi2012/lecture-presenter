# 常见问题（FAQ）

## 课程相关

### Q: 导入课程时提示"找不到课程配置"

**原因**：选择的目录中没有 `course.json` 文件。

**解决方法**：
1. 确认你选择的是课程的根目录（包含 `course.json` 的那个文件夹）
2. 不要选择课程内部的子目录（如 `slides/` 或 `videos/`）
3. 检查 `course.json` 文件是否存在且文件名拼写正确

---

### Q: 课程导入成功但看不到内容

**可能原因**：
- `course.json` 中的文件路径与实际文件位置不匹配
- 资源文件被移动或删除

**解决方法**：
1. 检查 `course.json` 中的路径是否正确
2. 确认路径是相对于课程根目录的
3. 确认对应的文件确实存在于指定位置

---

### Q: 如何删除已导入的课程？

1. 点击设置按钮 → **管理课程**
2. 找到要删除的课程
3. 点击删除按钮

> **注意**：删除操作只会从应用列表中移除课程引用，不会删除磁盘上的文件。

---

## 视频相关

### Q: 视频无法播放

**可能原因**：
- 视频格式不是 MP4 (H.264)
- 文件路径有误
- 视频文件损坏

**解决方法**：
1. 确认视频文件格式为 `.mp4`（H.264 编码）
2. 检查 `course.json` 中的路径是否正确
3. 尝试用其他播放器打开视频文件，确认文件完好
4. 如果是其他格式（如 MKV、AVI），请先转换为 MP4 格式

---

### Q: 视频播放卡顿

**解决方法**：
- 关闭其他占用大量 CPU/内存的应用
- 如果视频文件过大（超过 1GB），考虑压缩后再使用
- 确保视频文件存储在本地磁盘，而非网络驱动器

---

## PPTE 相关

### Q: PPTE 幻灯片显示空白

**可能原因**：
- `manifest.json` 格式不正确
- HTML 文件中有语法错误
- 文件路径引用有误

**解决方法**：
1. 检查 `manifest.json` 中的文件名是否与实际 HTML 文件名一致
2. 用浏览器直接打开 HTML 文件，检查是否正常显示
3. 确认 HTML 文件的编码为 UTF-8

---

### Q: 如何导出 PPTE 模板？

PPTE 模板本质上是 HTML 文件，你可以：

1. 找到课程目录下的 `ppt-extra/` 文件夹
2. 复制整个 PPTE 目录（包括 `manifest.json` 和所有 HTML 文件）
3. 粘贴到其他课程的目录中
4. 在 `course.json` 中添加对应的引用

---

### Q: 自己写的 HTML 幻灯片比例不对

**解决方法**：确保 CSS 中使用了标准的 16:9 比例代码：

```css
html, body {
    margin: 0;
    padding: 0;
    width: 100vw;
    height: 100vh;
    overflow: hidden;
    display: flex;
    align-items: center;
    justify-content: center;
}

.slide {
    width: min(100vw, 177.78vh);
    height: min(100vh, 56.25vw);
    max-width: 100vw;
    max-height: 100vh;
    padding: 60px;
    box-sizing: border-box;
}
```

## 应用设置

### Q: 配置文件存储在哪里？

演讲宝的配置文件存储在系统的应用数据目录中：

```
macOS: ~/Library/Application Support/com.lecture-presenter.app/
```

该目录下包含：
- `app-config.json` — 应用配置（课程路径、主题、字号等）

---

### Q: 如何重置应用设置？

1. 退出演讲宝
2. 删除配置目录中的 `app-config.json` 文件
3. 重新打开应用，会自动生成默认配置

---

### Q: 如何切换深色/浅色主题？

1. 点击标题栏的 **设置按钮**
2. 在主题选项中选择 **浅色** 或 **深色**
3. 主题会立即生效

---

## 性能相关

### Q: 应用启动很慢

**解决方法**：
- 检查导入的课程中是否有超大文件
- 减少同时导入的课程数量
- 确保应用安装在 SSD 上

---

### Q: 内存占用较高

**解决方法**：
- 浏览大型 PDF 或视频时内存占用会增加，这是正常现象
- 切换到其他资源后，之前的资源会被释放
- 如果持续过高，尝试重启应用

---
