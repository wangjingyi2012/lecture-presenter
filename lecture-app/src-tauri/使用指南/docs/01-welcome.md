# 欢迎使用演讲宝

## 你的智能课程管理与演示助手

**演讲宝（Lecture Presenter）** 是一款专为教学和学习场景设计的桌面应用，帮助你轻松管理课程资料、浏览多格式内容、制作精美幻灯片。

---

## 核心功能一览

### 1. 多格式课程浏览

演讲宝支持在一个统一的界面中浏览多种格式的教学资源：

- **PDF 课件** — 内置高质量 PDF 查看器，支持缩放和页码导航
- **视频教程** — 内置播放器，支持 MP4 格式
- **Markdown 文档** — 自动渲染为美观的富文本
- **代码文件** — 语法高亮显示，支持数十种编程语言
- **HTML 页面** — 完整的 HTML 渲染引擎
- **外部链接** — 一键在默认浏览器中打开

### 2. PPTE 幻灯片系统

PPT-EXTRA（PPTE）是演讲宝独创的幻灯片格式：

- 基于 HTML 技术，拥有无限的视觉表现力
- 支持渐变色、动画、自定义排版
- 内置多套精美模板，一键生成
- 支持 AI 辅助编辑，智能调整内容和样式

### 3. AI 智能助手

- 可配置 DeepSeek 或 MiniMax API Key
- 辅助 PPTE 幻灯片内容编辑
- 智能生成和优化幻灯片文案

### 4. 课程管理

- 轻松导入本地课程文件夹
- 支持多课程切换
- 拖拽排序管理课程列表
- 可视化课程创建与编辑
