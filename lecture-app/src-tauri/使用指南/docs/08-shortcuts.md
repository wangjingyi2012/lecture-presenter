# 快捷键大全

## 概述

演讲宝提供了丰富的快捷键，帮助你高效地浏览课程和操作应用。以下是完整的快捷键列表。

---

## 章节导航

| 快捷键 | 功能 |
|--------|------|
| `Cmd + 1` | 跳转到第 1 章 |
| `Cmd + 2` | 跳转到第 2 章 |
| `Cmd + 3` | 跳转到第 3 章 |
| `Cmd + 4` | 跳转到第 4 章 |
| `Cmd + 5` | 跳转到第 5 章 |
| `Cmd + 6` | 跳转到第 6 章 |
| `Cmd + 7` | 跳转到第 7 章 |
| `Cmd + 8` | 跳转到第 8 章 |
| `Cmd + 9` | 跳转到第 9 章 |

> **提示**：这些快捷键对应 `course.json` 中 `weeks` 的 `number` 字段。

---

## 视图控制

| 快捷键 | 功能 |
|--------|------|
| `Cmd + =` | 放大（增大字号 / PDF 缩放） |
| `Cmd + -` | 缩小（减小字号 / PDF 缩放） |
| `Cmd + 0` | 重置为默认大小 |

---

## PPTE 幻灯片控制

| 快捷键 | 功能 |
|--------|------|
| `←` 左方向键 | 上一张幻灯片 |
| `→` 右方向键 | 下一张幻灯片 |
| 鼠标点击页面左侧 | 上一张幻灯片 |
| 鼠标点击页面右侧 | 下一张幻灯片 |

## 小技巧

### 快速切换章节

如果你正在学习一个多章节的课程，使用 `Cmd + 数字` 可以快速在各章节之间跳转，无需使用鼠标点击侧边栏。

### 调整阅读体验

- 觉得文字太小？按 `Cmd + =` 放大
- 想要更多内容在视野中？按 `Cmd + -` 缩小
- 想恢复默认？按 `Cmd + 0` 一键重置

### PPTE 翻页技巧

在浏览 PPTE 幻灯片时，可以将页面想象为分左右两个区域：
- 点击**左半边**翻到上一页
- 点击**右半边**翻到下一页
- 或者直接使用方向键，更加方便
