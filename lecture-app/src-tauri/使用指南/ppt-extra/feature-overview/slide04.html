支持的资源格式

📄
PDF 文档
内置 PDF.js 高质量查看器，支持缩放、页码导航，流畅浏览课件与论文。
.pdf
🎥
视频播放
HTML5 内置播放器，支持进度拖拽、音量调节、全屏播放。
.mp4
💻
代码查看
Highlight.js 语法高亮，支持数十种编程语言，行号显示。
.py .js .ts .go .rs ...
📝
Markdown
自动渲染为富文本，支持标题、表格、代码块、列表等完整语法。
.md
🎨
PPTE 幻灯片
独创 HTML 幻灯片格式，16:9 标准比例，渐变、动画、无限创意。
ppt-extra/
🌐
HTML & URL
HTML 文件内置渲染，外部链接一键跳转浏览器。
.html & https://
